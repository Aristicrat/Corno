# Corno

Corno is a premium guitar/instrument tuning app with a calm iOS-inspired visual design, real-time pitch detection, and release-ready web + iOS workflows.

## Project Structure

- App source: `figma-make/corno`
- iOS project (Capacitor): `figma-make/corno/ios`

## Quick Start (from repo root)

```bash
cd "figma-make/corno"
npm install
npm run dev
```

## Build and Release Check

```bash
cd "figma-make/corno"
npm run release:check
```

## iOS (Xcode)

```bash
cd "figma-make/corno"
npm run ios:sync
npm run ios:open
```

For more app-level details, see:

- `figma-make/corno/README.md`
